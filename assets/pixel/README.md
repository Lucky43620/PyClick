Pixel art minimaliste (16x16) généré pour prototyper l'inventaire (assets non commités par défaut pour éviter les problèmes de binaires).

- Icônes: bois, pierre/minerai, herbes, pièces, lingot, cuir, cristal, potions (rouge/bleu), parchemin, pioche, épée, bouclier, anneau.

Les PNG sont générables localement via `python assets/pixel/generate_icons.py` puis copiables dans `assets/pixel/` pour vos builds. Ils servent de placeholders en attendant des assets finaux.
